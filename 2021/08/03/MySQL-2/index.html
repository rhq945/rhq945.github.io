“+”

Mysql中“+”只能用作运算符，不能够用于拼接：

  1. select 123+90 = 213

    对于两方都是数值，执行“+”，得到加法结果；

  2. select '123'+90 = 213

    对于一方是字符型，执行“+”，试图将字符型转化为数值，转化成功，得到加法结果；

    select 'jone'+90 = 90

    转化不成功，则将字符型转化为0，得到结果；

  3. 拼接需要用到函数CONCAT

    NULL

NULL和任何的操作数段拼接，得到结果都是NULL；

常见SQL函数

单行函数

  1. 字符函数

    • LENGTH(str) 获取参数的字节个数并返回；

    • CONCAT(str,str...) 拼接字符串；

    • UPPER(str) 将英文字符大写并返回；

    • LOWER(str) 将英文字符小写并返回；

    • SUBSTRstr,pos,len) 截取字符串：从str字符串的第pos个位置开始输出len长度个字符到结束；

      注意：索引从1开始，不是从0开始；

    • INSTR(str,substr) 返回子串（substr）在大字符串（str）中第一次出现的起始索引，没有则返回0；

    • TRIM：删除字符串头尾的所有‘a‘字符，若未声明删除的字符，则删除空格；

    • LPAD(str,len,padstr)：用指定字符实现左填充指定字符串长度；若不是填充则将其截断，剩下左边的；

    • RPAD(str,len,padstr)：用指定字符实现右填充指定字符串长度；若不是填充则将其截断，剩下右边的；
    • REPLACE(str,from_str,to_str)：将str中的from_str全部转换为to_str，将转换以后的str输出；

  2. 数学函数

    • ROUND(x,y): 返回x的四舍五入后的值，小数点后保留y位；先算绝对值再加正负号；
    • CEIL(x): 向上取整，返回>=该参数的最小整数；
    • FLOOR(x)：向下取整，返回<=该参数的最小整数；
    • TRUNCATE(x,y)：截断，返回该参数小数点后保留y位的值；
    • MOD(x,y)：取余

  3. 日期函数

    • NOW()：返回当前系统日期+时间

    • STR_TO_DATE(str,format)：将日期格式的字符转换成指定格式的日期

      format代表的是str的日期格式，使系统了解数字和年月日的对应关系；

      返回时将不同输入的日期转换为统一格式：%Y-%m-%d；

    • DATE_FORMAT(date,format)：将日期转换成字符

      返回时需要将输入日期转换为format代表的统一格式；

  • CURDATE()：返回当前系统日期

  • CURTIME()：返回当前系统时间

  • YEAR(date)、MONTH(date)/MONTHTIME(date)、DAY(date)、HOUR(date)、MINATE(date)、SECOND(date)

    获取指定日期的年、月、日、小时、分钟、秒；
